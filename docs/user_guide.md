# User Guide for Monome Norns Grid Interface

Welcome to the Monome Norns Grid Interface! This guide will help you get started with using the grid to create and manipulate musical patterns. Follow these simple steps to begin exploring the features of your grid.

## Getting Started

1. **Connect Your Grid**: Ensure your grid is connected to the Norns device. You should see a light burst on the grid, indicating it's powered and ready to use.

2. **Initialize the Interface**: When you start the app, the grid will automatically initialize and set up the necessary parameters for musical exploration.

## Grid Layout

- **[Rec]**: Located at the far left (x = 4, y = 1). Press this button to start or stop recording. When recording is active, any notes you play will be captured.

- **[Play]**: Located at button 6 (x = 6, y = 1). Use this button to toggle playback of the current pattern. Press once to start playback and again to stop.

- **[Clear]**: Located at the far right (x = 13, y = 1). Press this button to clear the current pattern, removing all recorded notes.

- **[Seq 1] to [Seq 4]**: These buttons (x = 7 to 10, y = 1) are reserved for future use to set transformation rules for pattern steps.

## Recording Notes

- **Play Notes**: While recording is active, press any key on the grid to record a note. The pitch is determined by the key's position, and the note is stored with a fixed velocity.

## Visual Feedback

- **LED Brightness**: The grid uses different brightness levels to indicate the state of each button. High brightness indicates active functions like recording or playback.

## Exploring Further

- **Experiment**: Try different combinations of recording and playback to create unique musical patterns.

- **Transformations**: Although not yet implemented, look forward to using the sequence buttons to apply transformations to your patterns.

Enjoy exploring the musical possibilities with your Monome Norns Grid Interface! Feel free to experiment and create your own unique soundscapes. If you have any questions or need further assistance, consult the community or additional resources available online.
